Introduction

Python is a high-level, general-purpose programming language. It emphasizes readability and a syntax that allows programmers to express concepts in fewer lines of code than might be used in languages such as C++ or Java.

The language's core philosophy is summarized in the aphorism: Readability counts.

"Beautiful is better than ugly. Simple is better than complex."

— The Zen of Python

History

Python was conceived in the late 1980s as a successor to the ABC language and first released in 1991. The creator, Guido van Rossum, aimed to make a language that was easy to read and use.

1991
Initial public release (version 0.9.0).
2000
Python 2.0 released — introduced list comprehensions and garbage collection.
2008
Python 3.0 released — breaking changes to improve consistency.

Features

Notable features include:

  • Indentation-based blocks — code blocks defined by indentation.
  • First-class functions and lambda expressions.
  • Comprehensive standard library (batteries included).
  • Cross-platform portability (Windows, macOS, Linux).

Example

Simple Python function (formatted with <pre> and <code>):

def greet(name):
    """Return a greeting for name."""
    return f"Hello, {name}!"

print(greet("World"))  # prints: Hello, World!
