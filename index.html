Introduction to CSS

What is CSS ?

CSS (Cascading Style Sheets) is a language used to control the look and feel of web pages. It works together with HTML:

  • HTML → provides the content and structure (like a skeleton).
  • CSS → adds style such as colors, fonts, layouts, spacing, animations (like clothes and skin).

🎯 Why do we use CSS?

  1. Separation of content and design → HTML handles content, CSS handles design.
  2. Reusability → A single CSS file can style multiple web pages.
  3. Consistency → Ensures a uniform look and feel across the entire website.
  4. Responsive design → Helps websites work on all devices (desktop, tablet, mobile).
  5. Faster development → Change once in CSS, apply everywhere.

👨‍🔬 Who invented CSS?

  • Invented in 1994 by Håkon Wium Lie and Bert Bos.
  • Goal: To separate styling from HTML content.

📌 Versions of CSS

  • CSS1 (1996) → First version, basic styling.
  • CSS2 (1998) → Added positioning, media types.
  • CSS3 (1999–2011) → Introduced modules like Selectors, Animations, Flexbox, Grid. (Still widely used today).
  • CSS4 → Not an official version, but modern features continue to be added in modules.
📱 Responsive Web Design (RWD)
  • Ensures websites adapt to all screen sizes.
  • Achieved using:
  • Media queries → @media screen and (max-width: 768px) { ... }
  • Flexible layouts → Flexbox, Grid.
  • Fluid images & percentages instead of fixed pixels.

    • 🏗 CSS Layouts

    • Normal Flow (default browser layout).
    • Box Model (margin, border, padding, content).
    • Positioning (static, relative, absolute, fixed, sticky).
    • Flexbox (1D layouts)..
    • CSS Grid (2D layouts)

    ✍️ Styling Methods

    1. Inline CSS → Inside an element (<p style="color:red">Text </p >).
    2. Internal CSS → In <style> tag within HTML.
    3. External CSS → In a .css file linked with tag (best practice).

    ⚡ In short:

    CSS makes websites beautiful, consistent, and responsive. Without CSS, all web pages would look plain like black text on a white background.
